# Suggestions - OVOS Personas Marketplace
Date: 2026-03-13

## Enhancements
- [ ] **Offline Mode**: Bundle fonts as local assets to align with OVOS's privacy-first, offline-ready mission.
- [ ] **Dynamic Previews**: Use a mock message bus visualization to show how the persona would "respond" based on its prompt (purely client-side simulation).
- [ ] **Advanced Filtering**: Categorize personas by "Technical", "Creative", "Historical", and "Search-Based" instead of just text search.
- [ ] **JSON Schema Validation**: Add a CI step using `ajv` or similar to validate `personas.jsonl` structure on PRs.
- [ ] **Custom Skin Builder**: Allow users to define their own `--accent` and `--bg` via the UI.
- [ ] **I18n Support**: Translate the UI and persona descriptions to match OVOS's multilingual community.
